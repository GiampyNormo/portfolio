# Edoardo Guttuso — Portfolio

Sito portfolio di Edoardo Guttuso, videomaker a Milano.
Sito statico (HTML/CSS/JS), senza dipendenze, pubblicato con GitHub Pages.

## Struttura
- `index.html` — home (presentazione)
- `portfolio.html` — gallerie video (orizzontali/verticali) + top brand
- `assets/` — `style.css`, `app.js` e **`progetti.js`** (la lista dei video)
- `media/` — foto, video di presentazione e loghi

## Aggiornare i contenuti
Vedi **COME-AGGIUNGERE-MEDIA.md**: per aggiungere/togliere video basta
modificare `assets/progetti.js` (i video dei lavori sono link YouTube).
